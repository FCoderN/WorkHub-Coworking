# WorkHub CoWorking

Sitio web para espacios de trabajo, permite explorar los espacios disponibles, conocer los servicios incluidos y realizar reservas en línea.

## Tecnologías

- HTML5
- CSS3 (variables CSS, animaciones, diseño responsivo)
- Bootstrap 5.3.2
- Bootstrap Icons
- Google Fonts (Playfair Display + Outfit)
- JavaScript

## Estructura
```
WorkHub-Coworking/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── app.js
└── recursos/
    ├── oficina1.jpg
    ├── oficina3.jpg
    ├── oficina5.jpg
    └── fondo.webp
```

## Secciones

- **Hero** — Presentación principal con llamados a la acción
- **Espacios** — Carrusel con tres opciones: escritorio individual, sala de reuniones y oficina privada
- **Servicios** — WiFi, café, impresión y estacionamiento
- **Formulario de Reserva** — Selección de espacio, fecha y hora
- **Footer** — Contacto, redes sociales y ubicación

## Cómo usar

Clonar el repositorio y abrir `index.html` en el navegador.
```bash
git clone https://github.com/FCoderN/WorkHub-Coworking
```

No requiere instalación de dependencias, todo se carga vía CDN.
